# Generator Rolek Wideo

Aplikacja webowa do automatycznego generowania krótkich rolek wideo z tekstem z pliku Excel.

## Funkcje

- Wczytywanie plików Excel z tekstami
- Dodawanie własnych filmów wideo
- Konfiguracja parametrów (długość klipów, pozycja tekstu)
- Automatyczne generowanie krótkich rolek
